# Stock_Analysis_Nifty50_Market_Trend
In this project we analysis of marketing trade of nifty50 stock data with visualization.

Here we have yaml file organized by moths.Each month folder contains date wise datas.So ,first extract that data from yaml and convert to 50 csv files based on Ticker.After that we calculate and visualize the trend.
1.Visualize the volatility of each stock over the past year by calculating the standard deviation of daily returns.Daily returns calculating formula is (close price-previous close price)/previous close price.Lower volatility is mentioned stable of stock trend.
2.Show the cumulative return of each stock from the beginning of the year to the end.Cumulative return calculated by running totals of daily returns.
3.We need to find the performance by sector wise by using bar chart.
4.Visualize the correlation between the stock prices of different companies.Calculated the correlation between the closing percentage of different stocks.
5.Finally we visualize the Top 5 Gainers and Top 5 Losers based on monthwise.
These 5 visualization makes on streamlit and plotted by matplotlib and powerbi.This nifty50 data will analysis very useful to understand the marketing trend of stock data.
